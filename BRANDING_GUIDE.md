# MetaIC AI - Branding Guide

## Logo Design

Based on the official MetaIC design:

### Logo Text: "MetaIC"
- **"Meta"** - Purple-pink gradient (#9333EA → #EC4899)
- **"IC"** - Yellow-green gradient (#FACC15 → #22C55E)
- Font: Sans-serif, bold
- Style: Modern, vibrant, AI-focused

### Color Palette

**Primary Colors:**
- Purple-Pink Gradient: For "Meta" portion
  - Start: #9333EA (Purple)
  - End: #EC4899 (Pink)
  
- Yellow-Green Gradient: For "IC" portion
  - Start: #FACC15 (Yellow)
  - End: #22C55E (Green)

**Background:**
- Dark purple-blue theme
- Hex: ~#1E1B4B or similar dark purple

**Accent Colors:**
- White: Primary text
- Light Gray: Secondary text
- Purple: Primary action buttons
- Dark Gray: Secondary buttons

## Icon Specifications

### App Icon
- Should reflect the gradient logo design
- "Meta" in purple-pink gradient
- "lC" in yellow-green gradient
- Dark background
- Modern, clean design

### Favicon
- Simplified version of logo
- Works at small sizes (16x16, 32x32)
- Maintains gradient colors

## Usage Guidelines

1. **Logo Placement:**
   - Centered on welcome/splash screens
   - Top-left on navigation bars
   - Prominent on landing pages

2. **Color Usage:**
   - Always use gradient colors for logo
   - Maintain contrast for readability
   - Dark theme preferred

3. **Typography:**
   - Sans-serif font family
   - Bold weight for logo
   - Clean, modern appearance

## Implementation Notes

- Logo should be SVG for scalability
- Support both light and dark themes
- Ensure accessibility (WCAG AA contrast)
- Maintain brand consistency across platforms
